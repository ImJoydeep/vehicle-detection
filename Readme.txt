

*************TO RUN THIS PROGRAM********************


1. Download python from www.python.org/downloads (if python interpreter is not downloaded)
2. Install python
3. Open cmd and type "pip install opencv-contrib-python"
4. Go to the folder path, eg:- "C:\Users\user\Desktop\demo>" and type "python main.py"

5.After running this program you can exit by pressing Esc. key.
